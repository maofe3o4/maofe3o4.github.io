IDEA使用

快捷方法

快速补全代码：ctrl+空格

表达式值类型转换：String s = ( 并按 Ctrl+Shift+空格 查看发生的情况

选择多个词语：shift+alt双击词语

查看类中实现接口的方法：光标置于类声明中的 implements 关键字处，然后按 Ctrl+Shift+F7

注释和取消注释: Ctrl+/用于单行注释(//…) Ctrl+Shift+/用于块注释(/* … */)

规范代码格式:选中要调整格式的代码ctrl+alt+L

查看类的继承关系:ctrl+H

定位方法属于哪个类:ctrl+B

自动分配对象变量名:通过在后面加.var 例如new 类().var

实时模板：file–setting–editor–live templates（比如fori直接生成for循环，main直接生成main方法）

运行.class文件：java 文件名

java程序执行入口：public static void main(String [] args){ }

类有属性和方法，将类实例化成对象后，该对象有对应的状态和行为

构造方法:与类名和类修饰一致，无返回类型，new对象时自动执行，一个类可定义多个构造方法(形成重载)

this的使用：哪个对象调用，this就代表那个对象

可变参数：int…a

对象和类
继承 一个类可以继承另一个类的属性和方法。
封装 将对象的状态私有，通过公共方法访问(类的变量私有，方法公共)
多态 通过方法重载和方法重写实现
抽象 抽象类和接口，不用具体实现
接口 定义类必须实现的方法，支持多重继承
方法重载 同一个类中有多个同名的方法，但参数不同。
方法重写 子类有一个和父类重名，重参数的方法，“重写父类方法”

instanceof运算符：检查该变量是否是一个特定类型的对象

运算符优先级：一元运算>四则运算>移位运算>关系运算>位运算>逻辑运算>赋值

java面向对象

封装

把数据和方法封装在一起，可以对数据进行验证

如果在构造方法里使用set方法，仍然可以对数据进行验证

继承

所有类都继承Object类，extends继承父类，implements继承(实现)接口，

final修饰表示该类不能被继承，或该方法不能被重写

多态

多态就是同一个接口，使用不同的实例而执行不同操作；多态的三个必要条件：继承、重写、父类引用指向子类对象。

  • 动物类 动物=new 狗() | 父类动物的引用指向了一个实际的狗对象（左边是编译类型，右边是运行类型）

  • 向上转型：将父类引用指向子类对象 父类引用 父类对象名=new 子类对象())

  • 向下转型: 将父类引用转为子类对象子类引用 子类对象名=(子类对象) 父类引用

  • 动态绑定机制：调用对象方法时，该方法和该对象的运行类型绑定

重写与重载

重写是子类与父类之间，有同名同参且返回类型也是父子或相同关系(如父类返回Object类型，子类返回String类型，String是Object的子类)的时候，子类对父类实现方法重写；

  • 不能重写父类构造器
  • 不能缩小父类方法的访问权限

重载是同一类中相同名字不同参数的方法实现重载

包

区分相同名字的类，实质就是文件夹，当同时调用两个不同包中相同类名的类时，应该加上包名加以区别。

开发者把一组类和接口等打包，并定义自己的包。将相关的类分组，更容易地确定哪些类、接口、枚举和注释等是相关的。
