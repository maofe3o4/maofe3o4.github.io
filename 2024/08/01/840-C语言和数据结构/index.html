C语言程序设计

概述

C语言特点
结构化控制语句 结构化程序设计
丰富的数据结构 复杂软件系统设计
允许直接访问物理地址 系统软件设计

标识符

  • 英文，数字，下划线
  • 长度1-32
  • 英文字母或下划线开头

语言风格

  • 大括号{}标识一个语句组，即一个复合语句

C程序结构

  • 一个或多个函数构成
  • 至少一个main的主函数
  • main函数可在任意位置

数据类型、运算符与表达式

转义字符
\n \t \b \r \f 换行 tab 退格 回车 换页
\ddd 1-3位八进制数代表的字符
\xhh 1-2位十六进制数代表的字符

字符串常量

  • 由一对双引号括起来的字符序列

运算符

  • 自增自减只能用于变量，不能用于常量
  • 逗号运算符，返回最后一个值
  • 加号优先级大于位运算

数据类型自动转换

char，short->int->unsigned->long->double<-float

数组

  • 赋初值，第一维长度可省略
  • 不能用赋值将字符串赋给字符数组，可以用strcpy(a,”abcd”)
  • 两个字符数组不能比较大小，可以用strcmp(a，b)，a>b返回1

函数

函数定义

  • 函数为定义返回类型，默认为int
  • 函数由函数首部和函数体组成
  • 函数之间允许直接递归调用也允许间接递归调用
函数存储类型
static 静态函数，只限于本编译单位中的其他函数调用
extern 外部函数，可省略，可被其他编译单位中的函数调用
变量
局部变量 auto动态存储区必须赋初值
static只赋一次初值，默认为0或空字符
register寄存器变量，执行更快		 函数形参属于局部变量，作用域定该变量的函数或复合语句
全局变量
  • 在内存中给用户使用的存储空间包括程序区，静态存储区和动态存储区
  • C语言的函数都是单值函数，return一个

main函数

输入输出，字符串函数

%符号
%〔标志〕〔输出最少宽度〕〔．精度〕〔长度〕类型

标志：
左对齐：”-“ “%-2s”代表左对齐2位
右对齐：”+” “%+2s”代表右对齐2位
不赋值给变量：*

printf的格式字符
d,i,u 带符号十进制，带符号十进制，无符号十进制
o 无符号八进制
x,X 无符号十六进制，大写
c,s 字符，字符串
f 浮点数
e,g 指数，f和e中谁短用谁
l 长整型，后面可加d，o，x，u
%m.nf 长度m，小数位n的格式输出；如果是字符串，则n标识截取的字符个数
- 左靠齐(不是负号)
scanf的格式字符
同上
* 表示本输入项读入后不赋给变量
  • scanf输入数据不能规定精度
  • 遇到空格，回车或者tab都会停止输入
函数
gets() 输入和输出可以包含\0
puts() 输入和输出可以包含\0
strcat(str1,str2) str1=str1拼接str2
strcpy(str1,str2) str1=str2拷贝2给1
strcmp(str1,str2) str1>str2返回正数，str1=str2返回0，str1<str2返回负数

预处理#

预处理语句包括宏，条件编译和文件包含

宏define

  • 宏替换是简单字符串替换，不包含括号
  • 宏不占运行时间，只占编译时间

条件编译

文件包含

  • <>直接到标准包含文件目录查找
  • “”先在当前目录查找，再到标准包含文件目录查找

指针

  • &取地址，* 取值(解引用)
  • 一个指针指向的变量的数据类型称为该指针的基类型
  • 数组a[i],中括号叫访问数据运算符，属于二目运算，访问以a为起点的第i个数据
  • [ ]的优先级高于 *

指针表示数组a[i] [j]的地址

  • a[i] +j
  • *(a+i)+j
  • a[0]+4*i+j 假设4列

函数指针

结构体和共用体

  • 定义时不占用存储空间，定义变量时才占用

定义结构体

引用结构体成员

  • 结构体变量名.结构体成员名
  • 结构体指针变量可以通过指针名->成员名

初始化

  • 结构体变量为全局变量或者静态变量，才能初始化
  • 结构体嵌套，初始化按顺序写初始值

共用体union

  • 不能在定义共用体变量时初始化
  • 共用体变量所占内存字节数与成员中最长的相等
  • 不能把共用体变量作为函数参数，但可以使用指向共用体变量的指针作为函数参数

枚举enum

  • 一个命名为整型常量的集合
  • 默认初始化从0开始

自定义类型typedef

位运算

位复合赋值运算：a<<=b 等同于 a=a<<b

位段

  • 将具有不同含义的多个小整数存放在一个机器字中
  • 如果一个字节剩下的空间不能容纳一个位段，则该位段放在下一个字节；上一字节的空位叫空穴
  • 位段没有名字表示该部分空间不用
  • 位段只能作为结构体成员，不能作为共用体成员，也不能定义位段数组

文件

  • C语言把文件看作一个字符序列
  • 根据数据组织形式，分为文本文件和二进制文件
  • 文本文件一个字节放一个ASCII码，文本结束标志0X1A
  • FILE类型是结构体类型，定义文件类型指针变量
  • C语言把所有外设都作为文件
常用外设 系统命名
键盘 KYBD或CON
显示器 SCRN或CON
打印机 PRN或LPT1
异步通信口 AUX或COM1

常用文件函数

mode：

  • r 、w、a、rb、wb、ab 读写追加(文本文件)，读写追加(二进制文件)
  • r+、w+、a+、rb+、wb+、ab+ 读再写、写再读、读再追加(文本文件)
函数 返回值 说明
fopen(fp，mode) 成功打开返回非0 以mode模式来打开文件
fclose(fp) 关闭成功放回0，否则返回-1 关闭文件并清除缓存区
fgetc(fp) 返回该字符，否则返回-1 从文件中读取一个字符
fputc(c,fp) 把一字节的字符c写入到文件里
fgets(str,n,fp) 读取到文件末尾，回车符，n-1个字符，读取结束 从文件中读取n个字符保存在str
fputs(str,fp) 出错返回EOF(-1) str写入到fp
fprintf(fp,format,arg1,arg2…) format和printf的格式符一致 把格式化的数据输入到文件中去
fscanf(fp,format,&arg1,&arg2…) 把文件中读取格式化的数据
fwrite(buf,size,count,fp) 从buf为首地址的内存中取count次数据，每次取size字节，写入到文件中
fread(buf,size,count,fp) 从文件中读count次数据，每次读size字节，存放到buf指定的内存去
ftell(fp) 获取当前文件指针的位置
fseek(fp,offset,from)
对于文本文件一般位移量为0		 起始位置0(文件开头)，1(文件当前位置)，2(文件末尾) 将文件指针移到从from位置开始位移offset字节处
rewind(fp) 将文件指针指向文件开头
