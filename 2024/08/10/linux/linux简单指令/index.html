常用简单指令

运行等级 说明
init 0 关机
init 1 单用户，root，用于系统维护，禁止远程登陆
init 2 多用户，不支持NFS
init 3 命令行模式，完全的多用户，登陆后进入
init 4 安全模式
init 5 图形化
init 6 重启

内存中的数据同步到磁盘：sync
查看当前目录：pwd
切换目录：cd 目录名 （cd只能进入目录）
进入文件：cat 文件路径
显示(当前)目录中的文件：ls [-a包括隐藏文件] [-l列表详细信息] [目录名]
创建目录：mkdir [-p] 目录名/路径
删除空目录(非空删除失败)：rmdir [-p] 目录名
清屏：clear
在终端使用之前用到的命令(历史命令保存在.bash_history文件中)：上下光标调取
命令自动补全：tab
正则表达式：*多个字符,?一个字符
查看/设置时间：date date -s “2024-01-02 09:11:42”
帮助指令：man help
修改主机名：hostname set-hostname 新名称

Linux目录和文件(简单介绍)

回到目录

绝对路径：/usr/include/a.h
相对路径：如果当前目录为/usr，则include/a.h为相对路径等同于上面
当前目录：.
上一级目录：..
家目录：~
系统启动必须：/boot、/etc、/lib
命令集合：/sbin、/bin
设备相关：/dev、/media、/mnt
临时文件：/run、/lost+found、/tmp
账户相关：/root、/home、/usr
运行过程：/var、/proc
拓展相关：/opt、/srv

bin执行，lib库，etc是配置，usr和home是’我的’，dev管设备，opt装软件

vi编辑器

回到目录

命令模式

命令模式 说明
:set nu 显示行号
:set nonu 取消行号
一般模式
/字符 查找指定字符，按n查找下一个
gg 查看行首
G 查看行尾
数字 G 查看指定行

i:当前位置开始编辑
o：在下面插入一行编辑
a：当前位置后一个位置插入
r：当前位置替换
I:当前行首个位置编辑
O：在上面插入一行编辑
A：当前行最后一个位置插入
R：从当位置开始替换

用户(组)管理

回到目录

组

组的信息保存在/etc/group文件
创建组：groupadd 组名
修改组：groupmod [-n 新组] 旧组
删除组：groupdel 组名

用户

创建删除
用户信息保存在/etc/passwd文件
用户密码保存在/etc/shadow文件
创建用户：useradd [-g 组名] [-d 用户主目录] [-r系统用户不可登录系统] 用户名
修改用户：usermod [-l 新名字] [-g 新组] [-d 新目录] 用户名
删除用户：userdel [-r删除组目录] 用户名
修改密码
root用户修改其他用户：passwd 用户名
普通用户修改自己：passwd
切换用户
root用户免密码切换：su -用户名
普通用户：su -用户名 然后输入密码
查看用户信息
id 用户名

目录文件操作

回到目录

目录和文件的详细信息：ls [-l] [目录名] 按时间降序显示ls -lt [目录名]
-a:列出全部文件，包括隐藏文件
快捷方式显示按当前目录下详细信息：ll

文件/目录属性

drwxr-xr-x. 4 root root 80 9月 26 17：05 aaa

对应信息 介绍 例子
文件类型 d目录-普通文件l链接p管道文件b块设备文件c字符设备文件s套接字文件 d
文件权限 r读w写x可执行-无权限；三个为一组代表所有者，同组用户，其他用户 rwxr-xr-x
文件硬链接数或目录的子目录数 文件硬链接，初始为1；空目录初始为2； 4
文件拥有者 root
文件拥有者所在组 root
目录或文件大小 文件大小单位字节；目录大小不包括目录下的文件和子目录 80
最后一次被修改的时间 创建/last修改 9月 26 17：05
目录或文件名 aaa

修改目录和文件权限

修改拥有者：chown [-R递归赋予权限] 用户：组 目录和文件列表
设置权限：chmod [-R] whooperatorpermission 目录和文件列表

who operator permission
u所有者 +增加权限 x进入目录或执行文件
g所有者所在组 -取消权限 r读取文件或查看目录中有哪些子目录和文件
o其他用户 =重新设置权限 w修改文件内容或在目录中增删改
a全部用户

r=4 w=2 x=1 rwx=4+2+1=7
chmod -R 777 文件名/目录 给所有人用给rwx权限

查看/查找文件内容

查看文件内容
cat 文件名 显示
less 文件名 分屏查看，
more 文件名 分页查看，空格下一页，b上一页，q退出
wc 文件名 统计文件行数
head [-n] 文件 查看前n行，默认10行
tail [-n] [-f] 文件 查看尾n行，默认10行,-f实时跟踪
grep [-n] 过滤查找， 内容|grep 指定查找内容
find 目录 -name 文件名 在指定目录下按名字查找文件
locate 文件名 locate之前updatedb

文件/目录的创建删除复制移动

创建目录/空文件 删除 复制 移动/重命名
mkdir [-mp] 目录名 rm [-rf] 目录和文件 cp [-r] 旧名 新名 mv 旧名 新名
touch 文件名 -r递归删除-f强制删除不提示 -r递归复制目录
-m配置权限；-p目录递归创建 rmdir是删空目录 \cp强制覆盖

在不同的linux服务器之间进行文件传输：
scp [-r] 文件 目标用户名@目标服务器名或ip:目标服务器上的保存路径

安装和打包

rpm包管理

命令 选项 说明
rpm -q 包名 查询是否安装
-qa 查询已安装的软件包
-ql 包名 查询软件包中文件
-qi 包名 查询软件包信息
-qf 文件全路径名 查询文件所属软件包
-e 包名 卸载包

yum安装

mv /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.backup

curl -o /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo

yum makecache

命令
yum [-y/d/n] install 包名 安装
yum -y remove 删除
yum update 包名 升级
yum list | grep 软件名 查找能够安装的包
yum check-update 显示可更新的包

打包和压缩

zip命令和tar命令

压缩：zip [-qrd] 压缩包文件名 目录和文件名列表
解压：unzip [-d 指定放的位置] 压缩包文件名

压缩：tar [-zcvf] 压缩包文件名.tar.gz 目录和文件名列表
解压：tar [-xzvf] 压缩包文件名 [-C 指定放的位置]

zip 说明 tar 说明
-q 不显示命令执行过程 -z 打包同时压缩
-r 递归压缩 -c/x 打包/解包
-u 把文件增加/替换到压缩包中 -v 显示文件详细信息
-d 指定压缩或解压缩后产生的文件放的地方 -f 指定压缩/解压的文件名
-t 列出压缩包里的文件(不用解压)
-C 指定压缩或解压缩后产生的文件放的地方

链接

回到目录

硬链接：ln 文件名 链接名
软链接：ln -s 目录或文件 链接名

硬链接是文件指针；软链接是一个保存了文件路径的文件

尽量为目录做软链接，为文件做硬链接

进程

回到目录

查看进程

ps [-aux]/[-ef]
选项说明：
-a 查全部进程(前台)
-x 查全部进程(包括后台)
-u 显示该进程的用户
-e 显示所有进程
-f 全格式

以树状形式显示
pstree [-p]/[-u]
选项说明：p：进程pid；u进程所属用户

实时显示top [选项]

选项 说明
-d 每各d秒更新显示内容
-i 不显示闲置和僵尸进程
-p 指定进程ID查看
执行后操作 说明
P 以CPU使用率排序(默认)
M 以内存使用率排序
N 以PID排序
q 退出top

第一行
top - 11：03：25 up 4 min 3 users load average：0.05,0.18,0.10
当前时间 运行了多久 当前登录用户数 系统负载：一段时间内，CPU正在处理和等待处理的进程数之和
上面三个数值分别是1分钟，5分钟，15分钟内的平均值
第二行: 统计进程 的统计 信息
Tasks:209 total 2 running 207 sleeping 0

终止进程

kill [-9] 进程号
kill all 进程名称 可用通配符

按端口查

lsof -t -i:端口号
-t只显示进程号
杀死某一端口的进程：kill $(lsof -t -i:端口号)

磁盘分区和挂载

回到目录

linux独特的挂载使得其只有一个根目录
查挂载情况-分区-分区格式化-挂载-查磁盘分区使用情况

分区方式 特点
mbr 最多支持4个主分区，系统只能在主分区，扩展分区占一个主分区，最大2TB，兼容性好
gtp 无限多个主分区(windows限制128个)，最大18EB
硬盘 说明
IDE 驱动器标识符hdx~ 例如hda3表示第一个IDE硬盘的第三个分区
SCSI sdx~
常用指令 选项 选项说明 说明
lsblk -f 会显示uuid和挂载点 查看所有设备挂载情况
fdisk /dev/sdb m p q w n m显示命令;p显示磁盘分区;q不保存退出;w写入并退出;n新增分区;e扩展分区 对/sdb分区,输入n,新增分区然后选择p ，分区类型为主分区。两次回车默认剩余全部空间。最后w写入分区并退出，或者q。
mkfs -t ext4 /dev/sdb1 -t 设置分区类型 对刚刚的主分区格式化
mount /dev/sdb1 被挂载的目录 -a -o 修改/etc/fstab后mount -a即可永久挂载 o选择何种方式挂载 不永久挂载重启会失效
-o ro 只读方式挂载
-o rw 读写方式挂载
umount 被挂载的目录/分区 取消挂载
df -h 带单位 查询磁盘、分区使用情况
du /目录 -s -h -a -c –max-depth=1 s占用大小汇总a含文件c列出明细同时增加汇总值max深度为1查看 查询指定目录的磁盘使用情况

防火墙

回到目录

命令 说明
systemctl start|restart|stop firewalld 启动、重启、关闭防火墙
firewall-cmd –zone=public –add-port=80-100/tcp –permanent 开通指定端口
也可以设置一个范围的端口80到100
firewall-cmd –zone=public –remove-port=80/tcp –permanent 关闭指定端口
firewall-cmd –zone=public –add-service=ftp –permanent 开通服务
firewall-cmd –zone=public –remove-service=http –permanent 移去服务
firewall-cmd –reload 重载

常用服务的端口：ssh(22）、HTTP(80、8080)、MySQL(3306)、Oracle(1521) 、ftp(21）、
telnet(23）。

查看已经开放端口
firewall-cmd –list-ports #查看已经开放的端口

环境变量（shell）

回到目录

输出内容

查看环境变量：env

命令 说明
command > file 将输出重定向到 file。
command < file 将输入重定向到 file。
command >> file 将输出以追加的方式重定向到 file。
n > file 将文件描述符为 n 的文件重定向到 file。
n >> file 将文件描述符为 n 的文件以追加的方式重定向到 file。
n >& m 将输出文件 m 和 n 合并。
n <& m 将输入文件 m 和 n 合并。
<< tag 将开始标记 tag 和结束标记 tag 之间的内容作为输入。
