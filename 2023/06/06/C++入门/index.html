一、指针

1.使用指针作为函数参数
主函数定义a，b，通过swap交换两值

2.使用const修饰指针
常量指针：const int *a 指向的内容不可变，不能用解引用修改地址中的值，一般用于函数形参，表示不希望在函数内部修改地址中的值。
指针常量（引用）：int * const a 指向的对象不可变，定义的同时必须初始化
常指针常量（常引用）：const int * const a 指向内容和对象都不可变
